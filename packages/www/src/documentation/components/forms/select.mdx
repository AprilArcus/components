The `<Select />` component renders a single select element on the page, with no accompanying label. It is generally used to construct higher-order components like the `<FieldText />`. If you are building a form, you probably want to use `<FieldSelect />` instead as it provides label and validation support.
By default `<Select />` is a combobox-type element, where the user can type in the field and also pick options from the menu via click or keyboard.
The input can be customized using `inputProps` like `placeholder`, `disabled`, and `readOnly` (for when you want the element to behave more like a native select).

For more customization, see [Composable Select](#composable-select) below.

```jsx
<Flex justifyContent="space-between">
  <Select
    options={[
      { data: 'cheddar', value: 'Cheddar' },
      { data: 'gouda', value: 'Gouda' },
      { data: 'swiss', value: 'Swiss' },
    ]}
  />
  <Select
    inputProps={{ placeholder: 'Placeholder Text' }}
    options={[
      { data: 'cheddar', value: 'Cheddar' },
      { data: 'gouda', value: 'Gouda' },
      { data: 'swiss', value: 'Swiss' },
    ]}
  />
  <Select
    inputProps={{ placeholder: 'Select one', readOnly: true }}
    options={[
      { data: '1', value: 'thing' },
      {
        data: 'Data can be used for extra identifying info',
        value: 'Value is rendered',
      },
      { data: '2', value: 'other' },
    ]}
  />
</Flex>
```

## useControlledSelect

Use the `useControlledSelect` hook to take control of your select's state.

```jsx
;() => {
  const selectControlProps = useControlledSelect('Cheddar')
  return (
    <Flex>
      <FlexItem mr="medium">
        <Select
          {...selectControlProps}
          options={[
            { data: 'cheddar', value: 'Cheddar' },
            { data: 'gouda', value: 'Gouda' },
            { data: 'swiss', value: 'Swiss' },
          ]}
        />
      </FlexItem>
      <FlexItem>Current value: {selectControlProps.value}</FlexItem>
    </Flex>
  )
}
```

## Composable Select

For ultimate customization, build your `Select` with `SelectInput`, `SelectList` and `SelectOption` children.

(This example is borrowed from [Reach UI's Combobox](https://reacttraining.com/reach-ui/combobox/) where much of our `Select` code comes from.)

```jsx
;() => {
  function Example() {
    const [term, setTerm] = useState('')
    // useCityMatch for demo purposes only
    const [results, debouncedTerm] = useCityMatch(term)
    const handleChange = event => setTerm(event.target.value)

    return (
      <Box>
        <Label>Lots of stuff going on</Label>
        <Select width="50%">
          <SelectInput
            onChange={handleChange}
            style={{ width: 300, margin: 0 }}
          />
          <SelectList>
            {results && results.length > 0 ? (
              <>
                <Heading as="h4" mx="medium" fontWeight="bold">
                  Top 3 results!
                </Heading>
                {results.slice(0, 3).map((result, index) => (
                  <SelectOption
                    key={index}
                    value={`${result.city}, ${result.state}`}
                  />
                ))}
                {results.length > 3 && (
                  <>
                    <Heading as="h4" mx="medium" fontWeight="bold">
                      The rest
                    </Heading>
                    {results.slice(3, 10).map((result, index) => (
                      <SelectOption
                        key={index}
                        value={`${result.city}, ${result.state}`}
                      />
                    ))}
                  </>
                )}
              </>
            ) : debouncedTerm.trim() === '' ? (
              <Box textAlign="center">Start typing to search cities</Box>
            ) : (
              <Box textAlign="center">
                <Paragraph mb="medium">No results ðŸ˜ž</Paragraph>
                <Button>Create a new record "{debouncedTerm}"</Button>
              </Box>
            )}
          </SelectList>
        </Select>
      </Box>
    )
  }

  return <Example />
}
```
