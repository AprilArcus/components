Components

Looker UI Components

A collections of assets for anyone making Looker software, providing Looker services, or telling Looker stories.

Install

Looker UI Components packages are served by NPM and can be installed through npm or yarn.

yarn add @looker/components

Explore

Search or browse through the component listing on this site. You can use the interactive prompts to compose whole components right in on this web site.

Contribute

Want something you don't see here? Follow the contribution guidelines and best practices, then open a Pull Request on the Github repository.
